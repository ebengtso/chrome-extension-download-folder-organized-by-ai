Chrome download organizer

Download Domain Router + Local AI

Manual domain rules first. Optional Chrome local AI or OpenAI classification for unmatched downloads.

Fallback behavior

Folder paths are relative to Chrome’s Downloads directory. Example: Law/Cases becomes ~/Downloads/Law/Cases on macOS.

AI classification

Not checked

AI cannot invent folders. Add folders here even when you do not yet have a domain rule for them.

Chrome local AI sends metadata to Gemini Nano running in Chrome, when available. OpenAI can optionally receive file content; Chrome local AI mode uses metadata only in this version.

Prompt template

Available placeholders: {{allowedFolders}}, {{allowedFoldersJson}}, {{defaultFolder}}, {{metadataJson}}, {{filename}}, {{hostname}}, {{url}}, {{mime}}, {{referrer}}.
